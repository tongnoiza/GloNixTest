
## Getting Started
```bash
1. ใช้คำสั้ง npm i เพื่อติดตั้ง Library package ของ โปรเจค
2. ใช้คำสั้ง npm run dev เพื่อ run โปรเจค
3. คลิกที่ลิ้ง [http://localhost:3000/GlonixTest) เพื่อเปิดหน้าเว็บบนบราวเซอร์                                        
```
